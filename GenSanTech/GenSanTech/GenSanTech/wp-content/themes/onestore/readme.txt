OneStore is a free WooCommerce WordPress theme which build & optimized for your online store. OneStore is lightweight, fast, highly customizable, and flexible theme and supports popular page builders like Elementor, KingComposer, Beaver Builder, SiteOrigin, Thrive Architect, Divi, Brizy, Visual Composer, etc. OneStore is compatible with YITH WooCommerce Wishlist, YITH WooCommerce Compare, WooCommerce Variation Swatches and other popular plugins. The theme is SEO friendly, WPML and RTL ready. (Live preview: https://sainwp.com/demos/onestore).

== Description ==

OneStore is a free WooCommerce WordPress theme which build & optimized for your online store. OneStore is lightweight, fast, highly customizable, and flexible theme and supports popular page builders like Elementor, KingComposer, Beaver Builder, SiteOrigin, Thrive Architect, Divi, Brizy, Visual Composer, etc. OneStore is compatible with YITH WooCommerce Wishlist, YITH WooCommerce Compare, WooCommerce Variation Swatches and other popular plugins. The theme is SEO friendly, WPML and RTL ready. (Live preview: https://sainwp.com/demos/onestore).

== Installation ==

1. Go to Appearance > Themes and click the Add New button.
2. Search for "OneStore" and wait until the theme appears.
3. Click Install Now button and then Activate the theme right away.
